# Tuitional AI Company Operating System

AI-powered company operating system for Tuitional Education and Tuitional AI.

## Architecture
See `/docs/ARCHITECTURE.md` for the master overview.

## Build Phases
See `/docs/BUILD-SEQUENCE.md` for the phased build plan.

## Tech Stack
- **Frontend**: React + Vite + TypeScript + Tailwind CSS + shadcn/ui
- **Backend**: Next.js API routes (Phase 2+)
- **Database**: PostgreSQL via Supabase
- **Agent Runtime**: OpenClaw v2026.3.7+
- **AI Model**: Anthropic Claude API

## Getting Started
```bash
npm install
npm run dev
```
